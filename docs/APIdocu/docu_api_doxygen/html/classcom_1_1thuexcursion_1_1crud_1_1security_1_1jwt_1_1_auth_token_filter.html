Detailed Description

This class is a filter that executes once per request. AuthTokenFilter class extends OncePerRequestFilter and override doFilterInternal() method. doFilterInternal() gets JWT from the Authorization header (by removing Bearer prefix) If JWT exists, doFilterInternal() validates it and parses username from it. From username, Userdetails is taken to create an Authentication object doFilterInternal() also set the current UserDteails in SecurityContext using setAuthentication(authentication) method
